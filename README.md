## Instructions

Esse projeto é referente ao módulo "JavaScript & DOM", do curso "Front End Web Developer", da Udacity.

O nome do projeto é "Landing Page" e tem como objetivo exercitar os tópicos abordados no módulo do curso.

Dentre as funcionalidades incluídas no projeto temos:

* A barra de navegação superior é criada com base na quantidade de seções existentes na página;
* Ao rolar a página a seção que está mais próximo ao topo fica destacada;
* Ao clicar em um botão da barra de navegação, a página é rolada até a seção correspondente, colocando-a
em destaque.
